# Advanced Portfolio

This folder contains a complete portfolio landing page scaffold.

## Files

- `index.html` — main portfolio page
- `styles.css` — responsive styling, dark/light theme, and layout
- `script.js` — mobile menu toggle, theme switching, reveal animations, and form stub

## How to use

1. Open `c:\Users\STUDENT\OneDrive\Desktop\styles\advanced-portfolio\index.html` in your browser.
2. Or use Live Server in VS Code to preview the page.
3. Customize the copy, project cards, email address, and branding.
